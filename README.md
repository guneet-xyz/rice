# rice

These are my is my personal configurations for software's and tools i use on across multiple machines.

## Usage

> Preface: You are encouraged to make your own adjustments to all the configurations. Ricing is a very personal endeavor.

All the configurations are stow-compatible. Read more about stow [here](https://www.gnu.org/software/stow/).

Example, to install the nvim config,

```
stow nvim
```
