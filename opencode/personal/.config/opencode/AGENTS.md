# Agent Instructions

## Skill Loading Protocol

Before starting ANY task:

1. List all files in the `/skills` directory (and subdirectories)
2. Read the filename and first few lines of each skill to understand what it covers
3. Load ALL skills that are even loosely relevant to the current request
4. Apply the patterns, conventions, and instructions from loaded skills to your work

## Important

- ALWAYS check for skills before writing code or making changes
- When in doubt, load the skill — it's better to over-load than miss relevant context
- If a task spans multiple domains (e.g., "build a contact form" = HTML + CSS + validation + API), load skills from ALL relevant domains
- Re-check skills when the task shifts direction mid-conversation
